# visual-physics-math
YouTubeチャンネル「目で見る物理・数学」の動画制作に使用した，数値シミュレーションおよびManimアニメーションのソースコード集です．

---

## コンテンツ一覧

興味のあるテーマのフォルダをクリックすると、数理的な背景とソースコードをご覧いただけます．

### 1. [エネルギー保存差分化（`energy-preserving-discretization/`）](./energy-preserving-discretization/)
* **概要**: 連続系の微分方程式を離散化（差分化）する際、エネルギー保存則が厳密に成り立つアルゴリズムを実装し、その挙動を可視化しました。
* **主な使用技術**: Python, NumPy, SciPy, Manim

### 2. [ニュートン法による最適化（`newton-method/`）](./newton-method/)
* **概要**: 方程式の解を近似計算する「ニュートン法」の収束プロセス、および初期値による挙動の変化を視覚的に解説するアニメーションです。
* **主な使用技術**: Python, NumPy, Manim

### 3. [振り子の等時性の破れ（`pendulum-period/`）](./pendulum-period/)
* **概要**: クイックソートやマージソートなどのデータ並び替えアルゴリズムの挙動と計算量の違いを、直感的に理解できるよう視覚化しました。
* **主な使用技術**: Python, Manim

---

## 開発環境・必要なライブラリ

本リポジトリのコードをローカル環境で実行するための共通環境です。

* **言語**: Python 3.10+
* **主要ライブラリ**:
  * `manim` (数学アニメーション生成エンジン)
  * `numpy` (高速数値計算)
  * `scipy` (科学技術計算)
